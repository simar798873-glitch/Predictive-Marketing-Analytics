# Predictive Marketing Analytics Using Machine Learning

## Overview

This project demonstrates how machine learning can be used to predict customer purchase behavior. 
The model analyzes customer data and determines whether a customer is likely to purchase a product or not.

This is a fundamental application of AI in marketing and business decision-making.

---

## Problem Statement

Businesses often struggle to identify which customers are most likely to purchase a product. 
Marketing to all customers increases cost and reduces efficiency.

This project solves this problem by building a predictive model that identifies potential buyers using customer data.

---

## Business Context

In real-world scenarios, companies collect customer data such as demographics, income, and website activity.

Using this data, businesses can:
- Target high-potential customers
- Reduce marketing costs
- Increase conversion rates
- Improve customer experience

---

## Objective

- Build a machine learning model to predict customer purchases  
- Understand predictive analytics in marketing  
- Apply classification techniques using real-world logic  
- Interpret model results for business use  

---

## Technologies Used

- Python  
- Google Colab  
- Pandas  
- Scikit-learn  

---

## Dataset Description

The dataset is created within the notebook for learning purposes.

Features used:

- Age: Customer age  
- Income: Monthly income  
- Website Visits: Number of visits to website  
- Time Spent: Time spent on website  
- Previous Purchases: Past buying behavior  
- Purchased: Target variable (1 = Yes, 0 = No)  

---

## Methodology

The project follows a structured machine learning workflow:

1. Data Creation  
2. Data Understanding and Exploration  
3. Feature Selection  
4. Train-Test Split  
5. Model Training using Logistic Regression  
6. Model Evaluation  
7. Prediction for New Customer  

---

## Model Used

Logistic Regression is used for this project because:

- It is suitable for binary classification problems  
- It is simple and easy to interpret  
- It performs well on structured datasets  

---

## Results

- The model generates an accuracy score to evaluate performance  
- It predicts whether a customer is likely to purchase or not  

Example Output:

- Accuracy: 0.85 (example)  
- Prediction: Customer is likely to purchase
